## About Project Disha

Project Disha is a six-round, team-based simulation in which student teams play the role of the Digital HR Transformation Office (DHTO) at GreenLeaf Retail — a 1,000-employee omnichannel retailer navigating a full digital-HR transformation.

### Key features
- Triple-currency resource model (Budget ₹ + Change Capital + Analyst Hours)
- Five named human anchor characters tracked across all rounds
- Two injected shock events: DPDP data breach (R3), viral gig-worker grievance (R5)
- Algorithmic fairness audit round (R4) with structured Fairness Audit Report
- Balanced-scorecard scoring with Equity & Compliance axis
- Persistent Debt Ledger tracking deferred problems
